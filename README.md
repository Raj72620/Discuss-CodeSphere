📦 Installation & Setup
Prerequisites
Node.js (v18 or higher)

MongoDB (Local or Atlas)

Git

🚀 Quick Start
Clone the repository

bash
git clone https://github.com/yourusername/discusshub.git
cd discusshub
Backend Setup

bash
cd backend
npm install

# Environment setup
cp .env.example .env
# Edit .env with your configurations
Frontend Setup

bash
cd ../frontend
npm install

# Environment setup
cp .env.example .env
# Edit .env with your API URL
Environment Configuration

Backend (.env):

env
MONGODB_URI=your_mongodb_connection_string
JWT_SECRET=your_jwt_secret_key
PORT=5000
Frontend (.env):

env
VITE_API_URL=http://localhost:5000
VITE_APP_NAME=DiscussHub
Run the Application

Start Backend Server:

bash
cd backend
npm run dev
Start Frontend Development Server:

bash
cd frontend
npm run dev
Access the Application

🌐 Frontend: http://localhost:5173

⚡ Backend API: http://localhost:5000
